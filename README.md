# TODO API
> Todo App API For Hepsi Emlak



## Installation
Prerequisites:
*  Docker
*  Docker Compose

```
docker-compose up -d
```

## Build/Package
```
mvn clean package
```

## Testing
```
mvn test
```

## Used Dependencies
* Spring Boot Web
* Spring Boot Validation
* Spring Boot Data Couchbase
* Spring Boot Security
* Springdoc OpenApi (openapi-ui)
* Lombok

## Abilities
* Couchbase Cache

## Swagger
> **Access : http://localhost:8080/swagger-ui/index.html#/
